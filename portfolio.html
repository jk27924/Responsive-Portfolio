PSYCHIC GAME

Psychic game is a type of Word Guess game. User earns Ten opportunities to guess only from alphabet a to z. When user's guess is right before using all Ten opportuities, the user wins. But, when user uses up all Ten opportunities, user loses and game restarts.

Technologies Used: HTML, CSS, Javascript

Crystal Collector

Crystal Collector is a guessing game uses numbers assigned to four different crystals. And by clicking the crystals, the user has to match the random number given since the beginning of the game.

Technologies Used: HTML, CSS, Javascript, jQuery
